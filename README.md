# Chefsito

Proyecto separado en dos aplicaciones:

- `chefsito-app`: frontend React + Vite + Tailwind.
- `server-chefsito`: backend Node.js + Express para la API de Ahorita.

## Frontend

```bash
cd chefsito-app
npm install
npm run dev
```

## Backend

```bash
cd server-chefsito
npm install
npm run dev
```

El backend usa `PORT=4000` por defecto.
